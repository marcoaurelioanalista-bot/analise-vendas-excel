# Dashboard de Análise de Vendas e Tickets no Excel

Este repositório contém uma planilha automatizada para gestão de clientes, valores de compras e status de tickets, desenvolvida para praticar e demonstrar funções avançadas de lógica e busca no Excel.

## 🚀 Funcionalidades Aplicadas
- **PROCV**: Localização automatizada de dados de clientes.
- **SE / E / OU**: Regras lógicas para definição automática do status de cada ticket.
- **SOMA / MÉDIA**: Indicadores financeiros consolidados.
- **SEERRO**: Tratamento de erros para manter a estética e integridade visual da planilha.

## 📂 Como usar
1. Baixe o arquivo `.xlsx` presente neste repositório.
2. Abra no Microsoft Excel ou Google Planilhas.
3. Insira novos IDs ou altere os valores para ver as fórmulas funcionando em tempo real.
