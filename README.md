# analise-vendas-excel
Mais um projeto prático concluído e documentado! Decidi aplicar algumas das funções mais utilizadas no mercado para construir uma ferramenta automatizada de análise de dados no Excel.Criei uma tabela estruturada com as colunas: ID, Cliente, Valor da Compra e Status do Ticket. Para trazer inteligência, automação e tratamento de erros à planilha.
